# Objetivo

Este projeto utiliza do framework Next.js e conceitos de Jamstack para a criação da página de uma empresa de alimentos matinais, com integração de APIs externas, preparação para deploy no CDN Netlify, importação de informações internas através do GraphQL e estilização básica.

## Dependências para este projeto
As seguintes versões foram utilizadas neste projeto:

- next@16.0.5
- npm@11.6.2
- react@19.2.0
- graphql@16.12.0

## Instalação inicial do projeto
1. Os arquivos package.json e jsconfig.json já estão pré configurados, instale as dependências necessárias com o comando abaixo no terminal:

```bash
npm i
```

## Executando o código

1. Rode o script de dev para criar os arquivos de produção e executar o servidor na porta http://localhost:3000

```bash
npm run dev
```
